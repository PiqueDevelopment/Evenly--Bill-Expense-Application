# bill-splitting
Bill splitting app, developed for Asha Hope Amanaki.

Input:
1. Description
2. Date
3. Total Amount
4. People

Output:
1. Unique ID
2. Description (wrap text)
3. Date
4. Total Amount
5. Who Paid
6. Contribution Split (Either in % or $ amount)
7. Balance Split (How much money one person is deducted / paid)
